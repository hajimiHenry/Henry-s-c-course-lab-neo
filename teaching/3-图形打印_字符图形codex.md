## 题目：图形打印（字符图形）

输入一个正整数 `n`（不超过 20）和一个字母字符，按照给定规律打印出由该字符组成的图形。

示例：

- 输入：`5 c`
- 输出：

```text
c
cba
cbazy
cbazyxw
cbazyxwvu
```

所有字母保证在 26 个大小写字母范围内，并且“减 1”时按字母表循环：`A` 的前一个字母是 `Z`，`a` 的前一个字母是 `z`。

---

## 一、需要用到的基础知识

- **字符与 ASCII 码**
  - 在 C 语言里，`char` 本质上是一个整数（ASCII 码）。
  - 可以对字符做加减运算，比如：`'c' - 1` 得到 `'b'`。
- **循环结构**
  - 外层循环：控制“第几行”。
  - 内层循环：控制“这一行要打印多少个字符”。
- **行长度的规律**
  - 第 1 行：1 个字符。
  - 第 2 行：3 个字符。
  - 第 3 行：5 个字符。
  - ……
  - 第 `i` 行：`2 * i - 1` 个字符。

---

## 二、观察输出规律

以示例 `n = 5, ch = 'c'` 为例：

- 第 1 行：`c`
  - 长度 1，从 `c` 开始。
- 第 2 行：`cba`
  - 长度 3，从 `c` 开始，每次往前一个字母：`c -> b -> a`。
- 第 3 行：`cbazy`
  - 长度 5，从 `c` 开始：`c b a z y`
  - 注意：`a` 再往前是 `z`（循环）。
- 第 4 行：`cbazyxw`
- 第 5 行：`cbazyxwvu`

总结两点规律：

1. 一共打印 `n` 行。
2. 每一行：
   - 从同一个起始字符 `ch` 开始。
   - 这一行打印 `2 * 行号 - 1` 个字符。
   - 每打印一个字符，就变成前一个字母，并且 `a` 之前是 `z`，`A` 之前是 `Z`。

---

## 三、整体思路（伪代码）

1. 读入 `n` 和字符 `ch`。
2. 外层循环：`i` 从 1 到 `n`，表示第 `i` 行：
   - 把当前字符 `cur` 设为 `ch`。
   - 计算这一行的长度 `len = 2 * i - 1`。
   - 内层循环：循环 `len` 次：
     - 打印当前字符 `cur`。
     - 让 `cur` 变成前一个字母（注意 a/A 的循环）。
   - 打印换行。

---

## 四、关键细节讲解

### 1. 输入格式

如果输入类似：`5 c`（数字和字符之间有空格或回车），推荐：

```c
scanf("%d %c", &n, &ch);
```

`%d` 读整数，` %c` 前面有一个空格，可以自动跳过空格、换行等空白字符。

### 2. 计算每一行的长度

第 `i` 行长度：`2 * i - 1`。

例如：

- `i = 1`：`2 * 1 - 1 = 1`
- `i = 2`：`2 * 2 - 1 = 3`
- `i = 3`：`2 * 3 - 1 = 5`

### 3. 字母向前推一位并循环

要实现：

- `b` 之前是 `a`
- `a` 之前是 `z`
- 对大写字母同理：`B -> A -> Z`

可以这样写逻辑：

```c
if (cur == 'a') {
    cur = 'z';
} else if (cur == 'A') {
    cur = 'Z';
} else {
    cur--;
}
```

---

## 五、完整参考代码（讲解版）

```c
#include <stdio.h>

int main(void)
{
    int n;
    char ch;

    // 读入一个整数和一个字符
    scanf("%d %c", &n, &ch);

    // 外层循环：第 i 行
    for (int i = 1; i <= n; i++) {
        char cur = ch;           // 每一行都从同一个字符开始
        int len = 2 * i - 1;     // 这一行要打印的字符数

        // 内层循环：打印这一行的所有字符
        for (int j = 0; j < len; j++) {
            putchar(cur);

            // 把 cur 变成前一个字母，注意循环
            if (cur == 'a') {
                cur = 'z';
            } else if (cur == 'A') {
                cur = 'Z';
            } else {
                cur--;
            }
        }

        putchar('\n');  // 一行结束换行
    }

    return 0;
}
```

## 六、自己练习的小建议

- 练习 1：把循环改成从 `ch` 开始，每次“加 1”而不是“减 1”，看看图形会变成什么样。
- 练习 2：试试不同的 `n` 和字符，例如：`3 Z`、`4 a`，自己推一下每一行应该是什么，然后运行程序验证。
- 练习 3：把行长度从 `2 * i - 1` 改成 `i`，观察输出图形的变化。

多画几行、手算几步，会更容易看懂“行数”“行长度”和“当前字符变化”之间的关系。
