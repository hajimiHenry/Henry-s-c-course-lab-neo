# 4_最大值与格式化输出codex

## 题目概述
有 `N` 个投保人，每个人当年投保了若干次（次数不等）。需要统计**每个投保人的最高投保金额**，并按“**万元**”为单位输出，保留 **2 位小数**。

---

## 输入/输出理解
### 输入格式
- 第 1 行：正整数 `N`（投保人数，`N ≤ 7`）。
- 接下来 `N` 行：每行第一个整数 `n`，表示该投保人当年投保次数（`n ≤ 10`），后面跟着 `n` 个正整数，表示每次投保金额（单位：元，每次不超过 100 万）。

### 输出格式
- 输出 `N` 行，每行一个实数（`%.2f\n`），表示该投保人的**最高投保金额**（单位：万元）。

### 样例解释
例如某人最高金额是 `125830` 元：
- 换算成万元：`125830 / 10000 = 12.583`
- 输出两位小数：`12.58`

---

## 知识点
### 1) “求最大值”的通用套路
对一组数求最大值：
1. 先准备一个 `max`（通常从 0 开始，因为金额是正数）
2. 逐个读入 `x`
3. 如果 `x > max`，就更新 `max = x`

### 2) 单位换算与浮点输出
- 题目要求“万元”，而输入是“元”
- 关键是要得到**小数**，所以要用浮点除法：
  - `max / 10000.0`（注意是 `10000.0`，不是 `10000`）
- 保留两位小数用 `printf("%.2f\n", value);`

### 3) scanf 读取“每行不定个数”
每行的结构是：先读次数 `n`，再循环读 `n` 个金额。

---

## 解题思路（按人处理）
对每个投保人：
1. 读入投保次数 `n`
2. 循环读入 `n` 个金额，求最大值 `maxMoney`
3. 输出 `maxMoney / 10000.0`，保留两位小数

时间复杂度：最多 `7 * 10 = 70` 次金额比较，基本就是 `O(总金额个数)`。

---

## 易错点清单
1. **忘记读每行第一个数 `n`**：每行金额个数不是固定的，必须先读 `n` 才知道后面读几个。
2. **用整数除法导致小数没了**：`maxMoney / 10000` 会变成整数（小数部分直接丢失）。
3. **输出格式不对**：题目要求每个人一行，且 `%.2f` 两位小数。
4. **变量类型选择**：
   - 金额最大 100 万（`1,000,000`），用 `int` 足够；
   - 习惯上也可以用 `long long` 更稳妥（尤其当题目范围变大时）。

---

## 参考实现（C 语言）
```c
#include <stdio.h>

int main(void) {
    int N;
    if (scanf("%d", &N) != 1) return 0;

    for (int i = 0; i < N; i++) {
        int n;
        scanf("%d", &n);

        long long maxMoney = 0;
        for (int j = 0; j < n; j++) {
            long long x;
            scanf("%lld", &x);
            if (x > maxMoney) maxMoney = x;
        }

        printf("%.2f\n", maxMoney / 10000.0);
    }
    return 0;
}
```
