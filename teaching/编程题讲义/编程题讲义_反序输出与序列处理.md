# 编程题讲义：反序输出与序列处理

> 目标：把“反序输出”归纳成三种套路：数组保存、递归、或双指针。

## 1. 反序输出为什么常见
它考的是：
- 能否把输入保存下来（数组）
- 能否正确处理下标边界
- 输出格式是否正确

## 2. 三种常用做法

### 2.1 数组保存（最常用、最稳）
适用于输入规模明确、能一次读完：
```c
for (i = 0; i < n; i++) scanf("%d", &a[i]);
for (i = n - 1; i >= 0; i--) printf("%d", a[i]);
```
注意：`i` 需要是有符号整型（`int`），否则 `i>=0` 可能出问题。

### 2.2 递归（常用于“不方便存数组”的题型）
思路：先递归读完再回溯输出。
```c
void read_and_print(int n)
{
    int x;
    if (n <= 0) return;
    scanf("%d", &x);
    read_and_print(n - 1);
    printf("%d", x);
}
```
注意：递归深度太大可能栈溢出，规模大时不建议。

### 2.3 双指针（适合字符串反转）
```c
int l = 0, r = len - 1;
while (l < r) { swap(s[l], s[r]); l++; r--; }
```

## 3. 输出格式细节
常见要求：
- 数字之间是否有空格
- 末尾是否需要空格
- 是否需要换行

建议：先确定“元素间分隔符”的规则，再写输出。

## 4. 自检清单
- `n=0/1` 时循环是否正确？
- `for (i = n-1; i >= 0; i--)` 是否会死循环？（`i` 必须是 `int`）
- 输出分隔符是否正确（最后一个元素不多输出空格）？
