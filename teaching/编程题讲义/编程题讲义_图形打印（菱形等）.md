# 编程题讲义：图形打印（菱形等）

> 目标：把“打印图案”统一成坐标/行列/空格数的计算问题，避免靠肉眼试错。

## 1. 两种主流思路
### 1.1 逐行计算“前导空格 + 星号数”
适合菱形、三角形等对称图形。

例如菱形可以拆成上半部分与下半部分：
- 上半：空格递减、星号递增
- 下半：空格递增、星号递减

关键是找规律（用行号 `i` 表达空格和星号个数）。

### 1.2 双层循环 + 坐标条件
把输出区域看成网格 `(i, j)`，用 `if` 判断该位置打印什么。

## 2. 逐行法的模板（推荐优先用）
假设总高度是 `2*n-1`（上 n 行，下 n-1 行）：
```c
/* 上半部分：i = 0..n-1 */
for (i = 0; i < n; i++) {
    spaces = n - 1 - i;
    stars  = 2 * i + 1;
    /* 先打印 spaces 个空格，再打印 stars 个字符 */
}

/* 下半部分：i = n-2..0 */
for (i = n - 2; i >= 0; i--) {
    spaces = n - 1 - i;
    stars  = 2 * i + 1;
}
```

## 3. 输出细节
- 每行打印完一定要换行。
- 空格是 `' '`，不要用制表符。
- 注意题目是否要求最后一行也换行（多数平台接受，但以题目为准）。

## 4. 自检清单
- 图形是否上下对称？（上半与下半的规律是否一致）
- 最宽那一行的字符数是否正确？（通常是 `2*n-1`）
- 每行空格数是否为非负？（避免 `spaces` 计算错）
